Gluttonous Greg was my 2nd attempt at the 48 hour game jam, Ludum Dare.

The main goal of the game is to collect as many healthy food items as possible while avoiding the unhealthy burgers as the theme was "The more you have, the worse it is".

My main focus this jam was to write clean and well commented code while still having enough time to program the game mechanics. I accomplished this by making it straightforward to program the mechanics at the beginning by making it a very simple design.
